# Metin2-Quest-Deleter
**Created by blackdragonx61**

> g++9 -std=c++17 main.cpp QuestDeleter.cpp -lstdc++fs -s -o qdel

* Put qdel to game/share/locale/xx

* Set permissions to 777

* ./qdel questname
